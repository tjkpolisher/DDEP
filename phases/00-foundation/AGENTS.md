# AGENTS.md - Phase 00 Foundation

## 목적

프로젝트가 일관되게 동작할 수 있는 기본 구조, 실행 방식, 공통 스키마 위치, 개발 규칙을 세팅합니다.

## 범위

- Next.js + TypeScript 프론트엔드 초기 구조
- FastAPI + Python 백엔드 초기 구조
- `uv` 기반 Python 패키지 관리
- Docker Compose 로컬 실행 기준
- 공통 환경 변수 예시
- 린트/포맷/테스트 명령 정리
- 도메인 상수와 공통 타입 위치 결정

## 완료 조건

- 새 개발자가 README와 루트 `AGENTS.md`만 보고 로컬 실행 경로를 이해할 수 있습니다.
- Python 실행은 `uv run`으로 통일됩니다.
- 백엔드 의존성은 `pyproject.toml`과 `uv.lock`로 고정됩니다.
- 프론트엔드, 백엔드, DB의 로컬 실행 방식이 문서화됩니다.
- Phase 01 이후 작업자가 참조할 공통 데이터 모델 위치가 정해져 있습니다.

## 금지 사항

- 이 Phase에서 문항 DB를 대량 작성하지 않습니다.
- 진단 알고리즘을 완성하려고 하지 않습니다.
- 검색 에이전트 외부 호출을 구현하지 않습니다.
- 운영 배포 자동화를 과하게 만들지 않습니다.

## 산출물

- 프로젝트 루트 구조
- 실행 명령 문서
- 환경 변수 예시
- 기본 health check
- 공통 타입/스키마 초안
