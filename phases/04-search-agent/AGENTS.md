# AGENTS.md - Phase 04 Search Agent

## 목적

진단 결과의 취약 concept tag를 바탕으로 신뢰도 높은 학습 자료를 검색, 평가, 추천합니다.

## 범위

- concept tag를 검색 쿼리와 학습 목표로 변환
- 공식 문서, 공개 강의자료, 논문, 기술 블로그 후보 수집
- 난이도, 신뢰도, 최신성, 실습성, 중복도를 기준으로 스코어링
- 사용자 수준에 맞는 3~7개 자료 추천
- 추천 이유와 선행 개념 설명

## 소스 우선순위

1. PX4, ArduPilot, Betaflight, ROS/MAVLink 공식 문서
2. MIT, Stanford, ETH, KAIST 등 공개 강의자료
3. arXiv/IEEE abstract 기반 논문
4. 검증된 기술 블로그/튜토리얼
5. 커뮤니티 글은 트러블슈팅 참고용으로만 사용

## 스코어링 기준

기본 가중치는 다음을 시작점으로 둡니다.

- 신뢰도: 0.35
- 수준 적합도: 0.25
- 최신성: 0.15
- 실습성: 0.15
- 중복 제거: 0.10

## 완료 조건

- 취약 concept tag 입력으로 추천 자료 3~7개가 생성됩니다.
- 각 추천에는 URL, 제목, 출처 유형, 난이도, 신뢰도, 추천 이유가 포함됩니다.
- 검색 실패, 네트워크 실패, 낮은 신뢰도 결과만 있는 경우의 fallback이 정의됩니다.
- 외부 호출 비용과 rate limit을 고려한 캐시 또는 저장 전략이 문서화됩니다.

## 금지 사항

- 검색 결과를 검증 없이 그대로 사용자에게 노출하지 않습니다.
- 커뮤니티 글을 공식 학습자료와 같은 신뢰도로 취급하지 않습니다.
- LLM 요약 결과만으로 신뢰도를 결정하지 않습니다.

## 산출물

- 검색 쿼리 생성기
- 후보 수집 인터페이스
- 자료 스코어링 함수
- 추천 결과 스키마
- 실패/fallback 정책
