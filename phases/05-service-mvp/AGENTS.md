# AGENTS.md - Phase 05 Service MVP

## 목적

CAELUM 내부자가 실제로 사용할 수 있는 MVP 서비스 흐름을 완성합니다.

## 범위

- 내부 사용자 접근 방식
- 진단 시작/진행/완료 흐름
- 결과 저장과 재진단 비교
- 검색 에이전트 추천 자료 노출
- 기본 운영 로그
- Docker Compose 기반 로컬 또는 내부 테스트 배포

## 운영 기준

- 1차 사용자는 CAELUM 내부자입니다.
- 외부 공개 서비스 기준의 결제, 대시보드, 마케팅 페이지는 만들지 않습니다.
- 내부 테스트에서 문항 품질, 진단 흐름, 추천 자료 품질을 검증하는 데 집중합니다.

## 완료 조건

- 내부 사용자가 로그인 또는 초대 링크로 진단을 시작할 수 있습니다.
- 진단 결과와 추천 자료가 저장됩니다.
- 이전 진단과 최신 진단을 비교할 수 있습니다.
- 운영자가 최소한의 오류 로그와 사용 흐름을 확인할 수 있습니다.
- README에 내부 테스트 절차가 있습니다.

## 금지 사항

- MVP에서 결제/구독 기능을 구현하지 않습니다.
- 팀/기관용 관리자 대시보드를 이 Phase의 필수 기능으로 만들지 않습니다.
- 실제 운영 배포 전 개인정보, 외부 API 키, 로그 보존 정책을 문서화 없이 넘기지 않습니다.

## 산출물

- 내부 MVP 사용자 흐름
- 저장/재진단 비교 기능
- 추천 자료 표시 기능
- 운영 체크리스트
- 내부 베타 테스트 가이드
